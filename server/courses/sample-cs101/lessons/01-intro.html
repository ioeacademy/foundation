Lezione 1 — Cos'è un computer

Un computer è una macchina programmabile in grado di eseguire sequenze di istruzioni per elaborare dati. Le sue componenti principali sono:

  • CPU — unità di elaborazione
  • Memoria — conserva dati e istruzioni
  • Storage — conservazione persistente
  • I/O — ingresso/uscita verso l'utente o la rete

Anche il telefono che stai usando per leggere questa pagina è un computer.
